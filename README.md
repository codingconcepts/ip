# ip
Fetch your public IP from the terminal

### Installation

Find the release that matches your architecture on the [releases](https://github.com/codingconcepts/ip/releases) page.

Download the tar, extract the executable, and move it into your PATH:

```
$ tar -xvf ip_[VERSION].[OS].tar.gz
```

### Usage

```
$ ip
123.123.123.123
```
